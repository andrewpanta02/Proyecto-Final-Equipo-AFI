#README
1. este es un programa de transferencia de dinero de manera internacional, nos ayuda a verficar que la informacion del remitente y destinatario este correcta y asi evitar retrasos en la transaccion del retiro 
2. El programa pide los datos de la transferencia( remitente, destinatario, pais , monto)
3. Dependiendo el monto ingresdo, se hace el calculo de la tasa de cada pais
4. Y por ultimno te hace un resumen tipo un recibo, y luego lo guarda en un archivo.
